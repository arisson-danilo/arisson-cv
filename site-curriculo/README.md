# Currículo Online - Árisson Danilo

Este é um site de currículo médico minimalista criado com HTML5 e Tailwind CSS.

## Estrutura do Projeto

```
site-curriculo/
├── index.html          # Arquivo principal do site
├── assets/
│   └── profile.jpg     # Sua foto de perfil (substitua o placeholder)
└── README.md           # Este arquivo
```

## Como Usar

1. **Substitua a foto**: Coloque sua foto quadrada no arquivo `assets/profile.jpg` (substitua o arquivo placeholder)

2. **Personalize os links**:
   - Atualize o link do WhatsApp em `index.html` (linha ~47)
   - Atualize o link do Currículo Lattes (linha ~52)

3. **Abra o site**: Abra o arquivo `index.html` em qualquer navegador

4. **Deploy no GitHub Pages**:
   - Faça upload desta pasta para um repositório no GitHub
   - Ative o GitHub Pages nas configurações do repositório
   - Seu site estará disponível em `https://seu-usuario.github.io/nome-do-repo`

## Tecnologias Utilizadas

- **HTML5**: Estrutura semântica
- **Tailwind CSS**: Estilização via CDN
- **Lucide Icons**: Ícones vetoriais
- **Inter Font**: Tipografia moderna

## Design

- **Estilo**: Minimalista e limpo, com foco em confiança médica
- **Cores**: Azul marinho (navy) e branco
- **Responsividade**: Otimizado para mobile e desktop
